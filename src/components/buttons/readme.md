##The Dot button
###Demo
!!$Warm.buttons.Dot!!
###How to use it?
- First :
```
var Warm = require('Warm');
```

- And then just simply call:
```
<Warm.buttons.Dot />;
```

###What props can I use?
####Example
```
<Warm.buttons.Dot
    onClick={function () { alert("Dot button clicked!") }}
/>
```
####List of props
- onClick : specify a function to be called when button is clicked.
